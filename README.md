# 🚀 Bullet Dodger (Rust + Macroquad)

Um jogo de sobrevivência frenético desenvolvido em Rust usando o framework **Macroquad**. O objetivo é simples: desvie das balas vermelhas pelo maior tempo possível!

## 🎮 Funcionalidades
- **Dificuldade Dinâmica**: O jogo fica mais rápido e os inimigos aparecem com mais frequência conforme sua pontuação aumenta.
- **Sistema de Pontuação**: Ganhe pontos por segundo sobrevivido.
- **Game Over & Restart**: Tela de fim de jogo com opção de reiniciar instantaneamente pressionando `ESPAÇO`.
- **Movimentação Fluida**: Suporte para setas do teclado e teclas `WASD`.

## 🛠️ Tecnologias Utilizadas
- [Rust](https://www.rust-lang.org/)
- [Macroquad](https://macroquad.rs/) (Framework de jogos 2D)

## 🚀 Como Executar
1. Certifique-se de ter o Rust instalado.
2. Clone o repositório:
   ```bash
   git clone https://github.com/guivconB/rustGame.git
   ```
3. Entre na pasta:
   ```bash
   cd rustGame
   ```
4. Execute o jogo:
   ```bash
   cargo run
   ```

## 🕹️ Controles
- **Mover**: `Setas` ou `W`, `A`, `S`, `D`
- **Reiniciar**: `ESPAÇO` (após o Game Over)
